Confessio.ie enthält eine digitale Edition von Ludwig Bielers textkritischen Edition (repr. 1993).
Der lateinische Text, wissenschaftliche Kommentar und die Apparati und Beziehungen zwischen diesen wurden mit TEI modelliert.
Die digitale Edition wurde 2011 unter http://www.confessio.ie/etexts/confessio_latin#01 veröffentlicht.

Der TEI Code ist unter http://www.confessio.ie/more/downloads# (confessio_latin.xml) öffentlich zugängig.

Die Folien in diesem Ordner wurden für einen Kurzvortrag beim Arbeitstreffen der KONDE AG verwendet.
